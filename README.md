## 🧩 What this is

**skills** is a daily English coach for software engineers. It helps you sound more natural in work chats, meetings, code reviews, and one-on-ones.

It is for people who can already work in English but want better phrasing. The skill helps you move away from text-book English and use the kind of language teams use every day.

## 📝 Example phrases

Here are a few examples of the kind of language this skill can help with:

- Instead of: “I am unable to complete this by today”
  - Try: “I will not be able to finish this today”

- Instead of: “Please review this task at your earliest convenience”
  - Try: “When you have a moment, could you take a look at this?”

- Instead of: “I disagree with this idea”
  - Try: “I see the point, but I think we should consider another option”

- Instead of: “I have completed the work”
  - Try: “I have finished the work and shared the update”
